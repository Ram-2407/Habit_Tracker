# How to Run the Project

step 1: unzip
step 2: open Habit_Tracker folder in Vs code
step 3: open terminal (change it powershell to cmd)

	run in terminal
step 4: activate environment:
		.venv\Scripts\activate.bat
step 5: pip install django
step 6: pip install -r requirements.txt  
step 7: python manage.py migrate         
	
step 8: run server:
		python manage.py runserver
